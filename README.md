# app-Oder-Drink

- Bảo mật:
+ Đăng nhập 
+ Đăng ký 
+ Quên mật khẩu 

* Có 3 Role: Người dùng, nhân viên và quản trị

- người dùng: 
+ Đổi lại mật khẩu 
+ Chỉnh sửa hồ sơ người dùng 
+ Đăng xuất 
+ Tìm kiếm sản phẩm 
+ Thêm sản phẩm vào giỏ hàng 
+ Đặt hàng và chờ staff xác nhận 
+ Xem lịch sử đặt hàng 
+ Có thể hủy đơn hàng 

- Nhân viên:
+ Xác nhận đơn hàng
+ Xem doanh thu
+ Xem các sản phẩm bán chạy

- Quản trị:
+ Quản lý sản phẩm
+ Quản lý danh mục
+ Quản lý người dùng
